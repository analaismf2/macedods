# MACEDO Digital Solutions — Portfolio

Portfólio de Ana Laís Macêdo Fonte, Founder of MACEDO Digital Solutions.

## Como usar

Abra o arquivo `index.html` diretamente no navegador. Não precisa de servidor ou instalação.

## Personalizar

- **Email / GitHub / LinkedIn**: procure por `hello@macedods.com`, `github.com/macedods` e `linkedin.com/in/macedods` e substitua pelos seus links reais.
- **Logo**: substitua o bloco `.logo-mark` por uma tag `<img>` apontando para sua logo em PNG/SVG.
- **Links dos projetos**: nos cards de projetos, substitua `href="#"` pela URL real de cada projeto.
- **Percentuais de skills**: ajuste os valores `data-width` nas barras conforme preferir.

## Fontes utilizadas
- Syne (display/headings) — Google Fonts
- DM Mono (código/labels) — Google Fonts

## Tecnologias
HTML5 · CSS3 · JavaScript vanilla — sem frameworks, sem dependências.
